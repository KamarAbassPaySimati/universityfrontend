# EduAdmin Pro

A modern university management system built with React + Vite, featuring a clean admin interface for managing students, faculty, courses, finance, and academic operations.

## Features
- Student Management
- Faculty Management  
- Course Management
- Academic Records
- User Authentication
- Role-based Access Control

## Tech Stack
- React 18
- Vite
- Redux Toolkit
- Tailwind CSS
- Ant Design
